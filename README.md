В директории `tests` находятся тесты для проверки API, которое должно быть реализовано в веб-сервере.

Директория `web` содержит файлы фронтенда.

TODO-сервер на Go

О проекте:

Простой TODO-сервер на языке Go с использованием SQLite в качестве базы данных.
Проект позволяет создавать, читать, обновлять и удалять задачи (CRUD).
Основное назначение — отработка навыков работы с REST API, базой данных и Docker.

Выполненные задания со звёздочкой:
⭐ Реализован REST API для управления задачами: создание, получение, отметка как выполненной, удаление.
⭐ Интеграция с базой данных SQLite через пакет modernc.org/sqlite.
⭐ Настроен Dockerfile для сборки и запуска сервера в контейнере.
⭐ Реализованы переменные (TODO_DBFILE, TODO_PASSWORD, TODO_PORT)
⭐ Реализован процесс аутентицикации с помощью JWT-токена или пароля
⭐ Реализован search задач по дате, комментарию и заголовку

Инструкция по запуску кода локально:

Скопируйте базу данных (опционально) — при первом запуске создаётся новая БД в корне проекта

Запустите сервер:
go run main.go

Сервер слушает порт TODO_PORT (по умолчанию 7540)

URL для доступа: http://localhost:7540/

Авторизация
Тестовый пароль: 12345

JWT-токен действует 8 часов

Инструкция по запуску тестов

Запуск тестов:
go test ./...

Инструкция по сборке и запуску через Docker
Соберите образ из корня проекта:
docker build -t go-final-project .

Запустите контейнер командой:

docker run -p 7540:7540 final_project

Проверьте работу через браузер или curl:

http://localhost:7540/
